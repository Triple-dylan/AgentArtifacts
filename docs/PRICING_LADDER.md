# Pricing Ladder

| Tier | Price Range | Includes | Notes |
|---|---:|---|---|
| Free lead magnets | $0 | Partial packs, sample skill excerpts, template snippets | Capped scope; lead capture preferred |
| Entry prompt packs | $9-$29 | Focused prompt packs/chains | One-time default |
| Skills | $49-$199 | `SKILL.md` modules with schemas and error handling | Commercial baseline rights |
| Agents | $299-$999 | Packaged agents with configs, tests, deployment docs | Includes stronger implementation docs |
| Pro bundles | $999-$2999 | Multi-asset suites across categories | Core AOV driver |
| Enterprise/custom | $5000+ | Tailored packages, licensing, support | Contract and SLA driven |

## One-Time vs Subscription

- One-time is default for all SKUs.
- Updates-feed subscription is offered for:
- active provider compatibility updates
- trading connectors and schema updates
- premium changelog and migration guides
- Subscription can be purchased standalone or bundled as an add-on during/post-checkout.

## Pricing Policies

- Bundle discount targets: 15%-35% against sum of standalone prices.
- Cart threshold incentives: prefer bonus assets over blanket discounting.
- Limited-time discounts: 10%-20% max, 72-hour window, one redemption per user per bundle.
